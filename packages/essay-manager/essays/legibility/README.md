# Legibility Essay — Workspace

**Status:** Drafting (Section 1 done, Sections 2-6 TODO)

## Workflow

1. Voice dump into `voice-transcripts/YYYY-MM-DD-section-name.md`
2. Transcribe & clean into `raw-material.md`
3. Shape into `draft-v1.md` section by section
4. When all sections done → final edit pass → convert to MDX
5. Move final to `packages/blog-site/content/essays/legibility.mdx`
